      When the game is opened, this is the first page that the player will see. he has the option to choose between the 4 following tabs and each tab with what goal it will accomplish.

    1. Play

      2. Once the play button is clicked on, the user is showed the different levels of the game and each level needs to be played and finished before the userr can have access to the next.

    2. Help

      3. Once the help button is clicked on, there is no much the user can see than an userinterface giving hint about the game which is pretty much straightforward.

    3. High Scores

        4. Clicking on the High scores link will take the user to an interface that has 5 differents tab that he can navigate on.

      • Today

        • User is shown the scores of the people playing that game from the highest score to the lowest one.This example of image is blank as no player has already play it that day.

      • Last Week

        • User is shown the score that were recorded on the game platform the previous week from the highest to the lowest.

      • Last Month

        • User is shown the score that were recorded on the game platform last month from the highest to the lowest.

      • All Time

        • User is shown the list of every scores recorded on the game platform from the highest to the lowest.

      • My Scores

        • Here the user can actually see his actual scores.

    4. More Games

      5. In case the user is no longer interested in the archery game or want to try a different game; he can just click on "More Games" and he will be directed to the following page which contains differents kind of similar games.
